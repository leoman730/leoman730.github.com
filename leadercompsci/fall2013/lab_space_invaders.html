Project: Space Invaders

In this project you will be recreating a game similar to the game space invaders.

Monday: Getting started

  1. Open the project template:
    • Go to https://drive.google.com/file/d/0Bx12ANAPqJ7ZWG5sVXViM0V2TVU/edit?usp=sharing and download the template file.
    • Open the template file in BYOB.
    • Name the file 'space-invaders-YOUR-NAME' and save it in the folder on the desktop that says 'Save files here.'
  2. Add code to the player sprite so that:
    • When the green flag is clicked, the player goes to the x position of 0 and y position of -175.
    • When the right arrow is clicked, the player's x position changes by 10.
    • When the left arrow is clicked, the player's x position changes by -10.
  3. Add code to the bullet so that:
    • When the green flag is clicked, the bullet is hidden
    • When the space key is pressed.... 1) the bullet becomes visible and matches the position of the player
    • When the space key is pressed.... 2) the bullet starts moving up until it touches the top sprite
    • When the space key is pressed.... 3) becomes hidden after it touches the top sprite

Tuesday: Adding invaders

  1. Import the invader project template:
    • Open your project file (space-invaders-YOUR-NAME.ypr) in BYOB.
    • Import the invader template file into your project by downloading https://drive.google.com/file/d/0B5eA9XYPP3mQejlkTUhOc2U1Q1k/edit?usp=sharing, then importing it into your project by choosing "File" => "Import Project...".
  2. Following the instructions in the template, add code to the invader so that:
    • It starts at the top left corner of the stage and then moves back and forth across and down the stage..
    • It disappears if hit with a bullet.
    • It ends the game if it reaches the ground.
    3. Now, clone the one invader sprite to make 10 different invaders that move in unison!
  3. Once you've completed your lab, save your BYOB file as 'space-invaders-YOUR-NAME' and email it as an attachment to help@leaderscompsci.com

Wednesday: Catching up and Organizing

  1. Import the enemy sprite template from yesterday, and complete the directions if you haven't done so already.
  2. Organize your code by building your own blocks. Find code that you use more than once, and stash it in a command, better yet, a reporter or predicate. Your project must use at least one custom block.
  3. Once you've completed your lab, save your BYOB file as 'space-invaders-YOUR-NAME' and email it as an attachment to help@leaderscompsci.com

Thursday: Bug Fixes and Submitting!

  1. Today is the last day of the lab, so make sure you meet all the requirements from previous days and fix bugs in your code.
  2. Once you've completed your lab, save your BYOB file as 'space-invaders-YOUR-NAME' and email it as an attachment to help@leaderscompsci.com
