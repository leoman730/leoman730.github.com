Lab: Gorillas!

Project Overview

In this lab, we will be building the classic computer game Gorillas. If you've never played the game before, you can find an online version here: http://www.kongregate.com/games/moly/gorillas-bas.

  1. We'll be handing out a printed copy of the assignment, but you can also find it online here: https://docs.google.com/document/d/15hBB2QLA4Ln980txhhnHgeGBdg6GQ5KHU_TeNHqq6-g.
  2. To help you get started, here's a template project file containing the overall structure of the game, with some of the functions already written: http://docs.google.com/file/d/0Bx12ANAPqJ7ZNEhabVAzRWl3NEk.
  3. Once you've completed your assignment, make sure to save it and then email the BYOB file to help@leaderscompsci.com.
  4. Answer the following questions in your email:
    1. What is one thing that your were stuck on in lab this week that you later figured out?
    2. Name at least one thing (preferably a list of things) that you have learned in class recently (different from #1).
