Lab: Using Lists

In this lab, you will use your new knowledge of lists to build two functions: maximum of and average of.

Directions:

  1. Open up BYOB to create a new file.

  2. Create a block that will find the maximum number in a list, called maximum of. Make sure it's a reporter, in the operator category, and that it accepts one argument.

    Write the code inside of the block definition; it should be almost identical to the "mimimum" function that was constructed in class today. Just as a reminder, you will need to create two variables and your code will involve the repeat block.

    When you've written the block, make sure to test it in a variety of inputs. Can you think of special inputs that would be useful to test for experience points!

  3. Create a block that will find the longest word in a list, called longest word in . Make sure it's a reporter, in the operator category, and that it accepts one argument (list of words).

    Your code will be very similar to the maximum reporter that you created above, except you will use the 'length of' string operator in your comparisons.

  4. Create a block that will find the average of the numbers in a list, called average of. Make sure it's a reporter, in the operator category, and that it accepts one argument.

    Write the code inside of the block definition; it should be very similar to the minimum and maximum functions, but with code to sum up numbers instead of checking and setting values. Just as a reminder, you will need to create two variables and your code will involve the repeat block.

    When you've written the block, make sure to test it in a variety of inputs. Can you think of special inputs that would be useful to test for experience points!

  5. Once you've completed your lab, save your BYOB file and email it as an attachment to help@leaderscompsci.com.
