Lab: Function Parameters and Arguments

In this lab, you will use functions to build an intelligent spaceship cannon in Scratch. Topics in this lab: functions, parameters, arguments.

Directions:

  1. The lab has already been started for you here. Open the link to the Scratch project and log in to your account by clicking the 'Sign In' link at the top of the page.
  2. Once you are signed in, click the 'See Inside' button at the top right of the page to view the project code. Then click the 'Remix' button to start making your game from this project.
  3. Examine the contents of the computeAngleToEnemy block. What is this block doing?
  4. Now, create a function inside the pointAtAngle block for the spaceship sprite. It should use the argument passed in for "angle" to determine in which angle to point the spaceship.
  5. Next, create a function inside the fireAtAngle block that shoots the missile at the angle passed in as the "angle" argument.
  6. Finally, test out your code! If you've implemented the pointAtAngle and fireAtAngle blocks correctly, you should be able to move the enemy anywhere on the stage, and when you press spacebar, it should always fire in the direction of the enemy.
  7. (BONUS!) If you've already made it this far, write a new function fireAtAngleAndDistance that has two parameters: "angle" and "distance." With this function, the missile should only be shot the specified distance.
  8. Be sure to share your project by clicking the 'Share' button at the top right of the page so the project can be graded!
  9. Save time at the end of class to fill out his form and submit your lab.
