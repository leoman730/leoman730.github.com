Special Topics: Intro to Web Programming

Today you will create a website using HTML.

  1. Open TextEdit application
  2. Open new plain text document
  3. Copy this HTML code into your TextEdit document:
    <html>
    <body>
    <p>Hello World!</p>
    </body>
    </html>
  4. Customize your website
    • http://www.w3schools.com/html/default.asp
    • http://www.webmonkey.com/2010/02/html_cheatsheet
  5. Save your website as 'your_name.html' and email it as an attachment to help@leaderscompsci.com
  6. If you have extra time, check out the web course on codecademy: http://www.codecademy.com/tracks/projects
