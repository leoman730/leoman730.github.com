Special Topics: Intro to Web Programming (Part II)

Today you will add styles to your website using HTML and CSS.

  1. Open the W3Schools HTML editor at http://www.w3schools.com/html/tryit.asp?filename=tryhtml_intro
  2. Copy and paste your HTML file from yesterday into the text area on the left side of the screen.
  3. Click "Submit Code" to preview your web page and make sure it looks good.
  4. Now, customize your website by:
    • Adding an image (<img>), if you haven't already.
    • Adding CSS (hint: <style>...</style>), to add colors and custom fonts to your web page.
    • Extra hint: Find lots of custom CSS colors here:
  5. Copy and paste the HTML and CSS you wrote, and email it as an attachment to help@leaderscompsci.com
  6. If you have extra time, check out the CSS "image filter" properties here: http://html5-demos.appspot.com/static/css/filters/index.html. Once you have a set of filters that you like, copy the CSS over to your own web page to add those filters to your image.
  7. If you still have extra time, start working on this CSS course on Codecademy: http://www.codecademy.com/courses/css-coding-with-style/0/1
