Loops

This week, we'll be working with loops. By the end of the week, you'll be able to build programs that use loops and sound blocks to create unique synthesized music.

  1. Go to http://scratch.mit.edu/ and log in to your account.
  2. Start a new project. Using Sound blocks and the loop blocks in the "Control" category, create a song.
  3. Add a sprite to your stage. Using the Looks blocks, make the sprite dance in sync with the music.
  4. At the end of class, fill out this form to submit your lab: https://docs.google.com/forms/d/1LBGiZjIpK5PDD65Kz--J70ortAKwmcRO8FS30sCFMB4/viewform.
