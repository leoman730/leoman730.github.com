Variables I: Shapes

You've seen how to create gravity, and now you're going to experiment with broadcasting and receiving messages. They are an important mechanism for multiple sprites interacting.

  1. Go to http://scratch.mit.edu/ and log in to your account.
  2. Part 1: Create a variable named numberOfSides. Write a script that draws a shape with the number of sides set in numberOfSides. Your script should clear all previous pen marks, then draw a new shape with the correct number of sides. Hint: An exterior angle of a regular polygon has 360 / numberOfSides degrees.
  3. Experiment with your script to make sure it works by dragging the numberOfSides variable slider to new values and testing that it draws a shape with the correct number of sides.
  4. Part 2: Once you've finished Part 1, add a script to ask the user "How many sides would you like to draw?" and then drawing a shape with that number of sides.
  5. At the end of class, fill out this form to submit your lab.
