Using Functions

Functions are an important abstraction that save time in your programs, so let's practice building and using them!

  1. Go to http://scratch.mit.edu/ and log in to your account.
  2. Implement the shape drawing function that we built together in class today.
  3. Create a new "max of (number1) and (number2)" function that returns the maximum of two numbers. You will need to create a variable (ie. "max number") to store the result.
  4. Create a new "average of (number1) and (number2)" function that returns the average of two numbers. You will need to create a variable (ie. "average value") to store the result.
  5. At the end of class, fill out this form to submit your lab.
