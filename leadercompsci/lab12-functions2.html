Using Functions II

Today, you'll be making several small functions in order to become proficient with their creation and use.

  1. Go to http://scratch.mit.edu/ and log in to your account.
  2. Implement this function: It should make your sprite point left and move a variable number of steps.
  3. Implement this function: It should set a variable to "true" when the number argument is odd and "false" when it is even (ie "is 3 odd?" = true).
  4. Implement this function: It should set a variable to the square of the input value (ie. 2 squared is 4).
  5. Implement this function: This function is tricky. It should take a number as an argument, and the sprite should count down from the number to zero. You're not allowed to use any kind of repeat block.
  6. At the end of class, fill out this form to submit your lab.
