Project: Space Invaders, Part I

In this lab, you will use your new knowledge of functions to make some basic features of a Space Invaders game.

Directions:

  1. The project has already been started for you here. Open the link to the Scratch project and log in to your account by clicking the 'Sign In' link at the top of the page.
  2. Once you are signed in, click the 'See Inside' button at the top right of the page to view the project code. Then click the 'Remix' button to start making your game from this project.
  3. There are several pieces missing from the project. Click on each of the three sprites (spaceship, invader and missile) and you'll see everything you need to fill in marked with "FILL IN." Following the directions in each comment note, add blocks to make the game work.
