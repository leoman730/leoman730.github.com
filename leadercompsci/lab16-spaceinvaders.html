Project: Space Invaders, the COMPLETE project

Over the next three days, we will combine our work from the previous two labs to create a complete space invaders game.

Grading Rubric

  1. Stage setup: The stage and sprites should be fitting for the space invaders game theme.
  2. Player motion: The player should move left and right corresponding to the left and right arrow keys.
  3. Missiles: Missiles should fire when the player presses the spacebar and the missiles should destroy invaders.
  4. Invader grid: There should be a grid of invaders spawned when the game first starts.
  5. Invader motion: The invaders should move in a zigzag motion as described in lab 14.
  6. Winning/losing conditions: The game should end and "You Win!" should be displayed if all invaders are killed. If an invader touches you, the game should say "You Lose!"
  7. Code quality: Your code should make use of functions and variables where appropriate.

Directions:

  1. Log in to scratch and use your work from lab 14 or 15 (the last two labs) as the starting point for this project.
  2. Combine the work from the last two labs; it should already implement most of the features described above.
  3. Make sure to share your project and submit a link to it at the end of each day we work on it here.
