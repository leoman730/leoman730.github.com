Lab: Recursion

In this lab, you will use your new knowledge of recursion to write a function that computes the Nth Triangular Number.

Directions:

  1. Fork the following project in Scratch: http://scratch.mit.edu/projects/22456109/.
  2. The lab already has a "factorial" function filled in with what we went over in class. Using this function as a reference, create a NEW function named "triangular" which will compute the Nth triangular number.
